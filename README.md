# Proyecto Final SW

## Instalación de Python VENV
Crea una carpeta vacía en donde estará en VENV que vamos a crear, junto a la carpeta del proyecto
```sh
mkdir ProyectoFinal_SW
cd ProyectoFinal_SW

```

## Instalación de las dependencias

```sh
pip3 -r requirements.txt
```
